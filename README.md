# Python BMI Calculator

## Overview
Developed a Python-based Body Mass Index (BMI) calculator that computes and categorizes BMI based on user input, demonstrating programming logic and user interaction.

## Features

- Calculates BMI and categorizes it as Underweight, Normal, Overweight, or Obese
- Validates inputs to ensure they are positive numbers
- Provides clear, user-friendly output

## Skills Demonstrated
- Python programming and problem-solving
- Conditional logic and arithmetic operations
- User input handling and validation
- Translating health formulas into functional code
- Writing clean, maintainable, and well-documented code

## Tools Used
- Python (Jupyter Notebook)
- Standard Python libraries

## How to Use
1. Clone the repository or download the script.
2. Run `python bmi_calculator.py`.
3. Enter your weight in kilograms and height in meters when prompted.
4. View your BMI value and category.

## Outcome
Delivered a functional, interactive BMI calculator that educates users on their BMI category and encourages health awareness.
